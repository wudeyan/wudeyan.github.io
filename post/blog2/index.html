hugo个人博客搭建

环境搭建及配制

1.首先需要注册一个GitHub社区账号，并在GitHub社区上创建一个仓库

仓库的命名必须是以

"Username.github.io"

2.两个工具：

1）git工具，Windows系统下使用powershell的scoop工具进行下载

//安装scoop
Set-ExecutionPolicy RemoteSigned -scope CurrentUser
iex (new-object net.webclient).downloadstring('https://get.scoop.sh')
//下载速度过慢，可以利用aria2加速
scoop install aria2
//下载git工具
scoop install git

安装好git之后必须对git进行用户配置

git config --global user.name "Username"
git config --global user.email "xxxx@xxx.com"

为了方便使用，建议配置环境变量

2)进入GitHub社区搜索hugo，下载Windows版本的hugo（32位，64位自选）

https://github.com/gohugoio/hugo/releases

同样下载解压之后，为方便使用，先配置环境变量

博客搭建

1.进入hugo根目录，打开命令行界面，输入

//验证hugo时候已经安装
hugo version

2.新建一个站点

hugo new site myblog

发现会在根目录下创建一个myblog的文件夹

cd myblog

进入该博客目录

3.给自己的博客下载一个主题，可以到GitHub社区当中搜索

https://themes.gohugo.io

例如主题：m10c

点开主题会介绍下载的命令

//下载此主题
git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

主题会保存在/myblog/themes/目录下

4.在根目录下的content/post的创建一个博客

hugo new post/blog.md       //.md即markdown文件

博客中添加你的内容

5.尝试在本地启动hugo博客，并浏览博客

hugo server -t m10c --buildDrafts

6.将自己的博客与自己的远程GitHub关联

hugo --theme=m10c --baseUrl="https://wudeyan.github.io/" --buildDrafts

关联之后会在博客根目录下面创建一个public目录，与博客有关的所有文件都在此目录中

7.将此目录中的所有文件推上你的GitHub社区的仓库中

cd public

git init

git add .

git commit -m "我的 hugo 博客第一次提交"

git remote add origin https://github.com/wudeyan/wudeyan.github.io.git

git push -u origin master

8.浏览器中输入https://wudeyan.github.io浏览博客

其他命令

git pull –rebase origin master

git remote rm origin

git remote add origin git@github:bx_reader/bx-reader-api.git

…或在命令行上创建一个新的存储库

echo“#wudeyan.github.io”>> README.md 
git init 
git add README.md 
git commit -m“first commit” 
git remote add origin https://github.com/wudeyan/wudeyan.github.io.git
 git push -u origin master

…或从命令行推送现有存储库

git remote add origin https://github.com/wudeyan/wudeyan.github.io.git
 git push -u origin master
